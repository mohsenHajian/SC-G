Sauvegardes et réglages précédents

Stronghold Crusader HD mémorise les cartes, sauvegardes et réglages dans le fichier Documents\Stronghold Crusader. Si vous êtes passé d'une ancienne version de Stronghold Crusader à Stronghold Crusader HD, vous aurez besoin de copier les cartes, les sauvegardes et les réglages qui se trouvent dans le fichier Program Files\Firefly Studios\Stronghold Crusader (emplacement par défaut) au fichier Documents\Stronghold Crusader. 

Sauvegardes (fichiers .sav)

Ancien emplacement : Program Files\Firefly Studios\Stronghold Crusader\saves
Nouvel emplacement : Documents\Stronghold Crusader\Saves

Cartes (fichiers .map personnalisés)

Ancien emplacement : Program Files\Firefly Studios\Stronghold Crusader\maps
Nouvel emplacement : Documents\Stronghold Crusader\maps

Réglages (crusader.cgf)

Ancien emplacement : Program Files\Firefly Studios\Stronghold Crusader
Nouvel emplacement : Documents\Stronghold Crusader

Piste d'escarmouche (skmasters.dat)

Ancien emplacement : Program Files\Firefly Studios\Stronghold Crusader
Nouvel emplacement : Documents\Stronghold Crusader

REMARQUE : si vous utilisez Windows 64 bit, l'emplacement de l'installation par défaut est Program Files (x86)\Firefly Studios\Stronghold Crusader

Partie 1 – Crusader Extreme

Si vous découvrez Crusader, nous vous conseillons de jouer d'abord à la Piste du Croisé ou la Série d'escarmouches 2 avant de vous lancer dans la Série d'escarmouches Extreme. La Série d'escarmouches Extreme a été créée pour les joueurs expérimentés des titres Stronghold. Revêtez votre armure et que la bataille commence !

Stronghold Crusader Extreme propose deux versions de Stronghold Crusader : Stronghold Crusader Deluxe et Stronghold Crusader Extreme.

Stronghold Crusader Deluxe n'était sorti qu'aux Etats-Unis. Il propose tout le contenu du jeu Stronghold Crusader original, mais ajoute aussi de nouveaux adversaires à affronter et la Série d'escarmouches 2.

Stronghold Crusader Extreme est une toute nouvelle version de Stronghold Crusader, proposant de nouveaux éléments extrêmes, comme les avant-postes et les super attaques, utilisables aussi bien en multijoueur que dans les modes d'escarmouche. Il y a aussi de nouveaux adversaires à affronter et la Série d'escarmouches Extreme.

Partie 3.  Aide Multijoueur

Multijoueur Stronghold Crusader/Stronghold Crusader Extreme

Si votre ordinateur est connecté à un routeur ou un pare-feu, vous devez ouvrir les ports du pare-feu Microsoft DirectPlay suivants afin de pouvoir jouer à Stronghold Crusader ou Stronghold Crusader Extreme en mode Multijoueur.

 

 Sortant Entrant
Connexion TCP initiale 47624 47624
TCP suivant 2300-2400 2300-2400
UDP suivant 2300-2400 2300-2400


Si vous avez besoin d'aide pour configurer votre routeur, veuillez vous rendre sur portforward.com qui dispose d'un grand choix de guides sur plus de 1000 routeurs différents.

http://www.portforward.com/english/routers/port_forwarding/routerindex.htm

Pare-feu Windows

1.    Allez dans Panneau de configuration > Pare-feu Windows
2.    Cliquez sur l'onglet Exceptions
3.    Si Stronghold Crusader et Stronghold Crusader Extreme n'apparaissent pas dans la liste, cliquez sur Ajouter un programme
4.    Sélectionnez Stronghold Crusader puis cliquez sur OK
5.    Répétez le processus pour Stronghold Crusader Extreme
6.    Cliquez sur OK pour fermer la fenêtre Pare-feu Windows



Partie 4. - Caractéristiques supplémentaires

Mini-carte

En cours de partie, vous pouvez appuyer sur ALT-R pour afficher les ennemis d'une seule couleur et les alliés d'une autre sur la mini-carte.

Pouvoirs tactiques autour du seigneur

Dans le menu des options multijoueur Extreme, vous pouvez activer l'option Pouvoirs tactiques autour du seigneur. Cette option vous permet d'utiliser la barre d'Aide tactique pour faire apparaître des unités autour de votre seigneur et non plus uniquement autour de votre donjon.
