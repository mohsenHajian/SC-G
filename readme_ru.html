Добро пожаловать в новое издание Stronghold Crusader HD! Это издание поддерживает широкоэкранные разрешения до 2560 х 1600, что позволяет вам насладиться играми как никогда.

Надеемся, что вам понравится играть в HD-версии, и спасибо за поддержку.

Firefly Studios

Старые сохранения и настройки

Stronghold Crusader HD сохраняет пользовательские карты, сохранения и настройки в папке Documents\Stronghold Crusader. Если вы делали апгрейд до Stronghold Crusader HD со старой версии Stronghold Crusader, вам придётся скопировать карты, сохранения и настройки из папки Prgoram Files\Firefly Studios\Stronghold Crusader в папку Documents\Stronghold Crusader.

Сохранённые игры (.sav файлы)

Старое положение: Program Files\Firefly Studios\Stronghold Crusader\saves
Новое положение: Documents\Stronghold Crusader\saves

Карты (.map файлы) )

Старое положение: Program Files\Firefly Studios\Stronghold Crusader\maps
Новое положение: Documents\Stronghold Crusader\maps

Настройки (crusader.cfg)

Старое положение: Program Files\Firefly Studios\Stronghold Crusader
Новое положение: Documents\Stronghold Crusader

Выберите тип матча Crusader (skmasters.dat)

Старое положение: Program Files\Firefly Studios\Stronghold Crusader
Новое положение: Documents\Stronghold Crusader

Внимание: Если вы используете 64-битную версию Windows, то стандартной папкой для установки будет Program Files (x86)\Firefly Studios\Stronghold Crusader

Раздел 2 - Помощь для многопользовательского режима


Если ваш компьютер подсоединен к роутеру/брандмауэру, для того чтобы играть в "Stronghold: Crusader" или "Stronghold: Crusader Extreme" в многопользовательском режиме, необходимо разрешить соединение со следующими портами Microsoft DirectPlay:
 

 исходящие входящие
Начальное TCP-подключение 47624 47624
Последующие TCP 2300-2400 2300-2400
Последующие UDP 2300-2400 2300-2400


Если вам требуется помощь при конфигурировании роутера, пожалуйста, посетите сайт, portforward.com, где представлены подробные руководства по настройке более чем 1000 роутеров.

https://portforward.com/router.htm

Брандмауэр Windows

1.    Откройте Панель управления > Брандмауэр Windows
2.    Выберите закладку Исключения
3.    Если игр "Stronghold: Crusader" и "Stronghold: Crusader Extreme" нет в списке, нажмите кнопку Добавить программу.
4.    Выберите "Stronghold: Crusader" и нажмите OK
5.    Повторите то же самое для " Stronghold: Crusader Extreme".
6.    Нажмите OK, чтобы закрыть окно Брандмауэра Window.
