Old savegames and settings

Stronghold Crusader HD stores user maps, saves and settings in the Documents\Stronghold Crusader folder.  If you have upgraded to Stronghold Crusader HD from an old version of Stronghold Crusader you will need to copy the maps, saves and settings from the Program Files\Firefly Studios\Stronghold Crusader folder (default location) to the Documents\Stronghold Crusader folder.

Savegames (.sav files)

Old Location: Program Files\Firefly Studios\Stronghold Crusader\saves
New Location: Documents\Stronghold Crusader\saves

Maps (custom .map files)

Old Location: Program Files\Firefly Studios\Stronghold Crusader\maps
New Location: Documents\Stronghold Crusader\maps

Settings (crusader.cfg)

Old Location: Program Files\Firefly Studios\Stronghold Crusader
New Location: Documents\Stronghold Crusader

Skirmish Trail (skmasters.dat)

Old Location: Program Files\Firefly Studios\Stronghold Crusader
New Location: Documents\Stronghold Crusader

NOTE: If you are using 64 bit Windows the default installation location is Program Files (x86)\Firefly Studios\Stronghold Crusader

Section 1 – Crusader Extreme

If you are new to Crusader, we recommend playing either the Crusader or Warchest trails before attempting the Crusader Extreme trail. The Crusader Extreme trail has been created for veterans of our Stronghold titles. Gird your loins and let battle commence!

Stronghold Crusader Extreme features 2 versions of Stronghold Crusader: Stronghold Crusader Deluxe and Stronghold Crusader Extreme.  

Stronghold Crusader Deluxe was previously only released in the US.  It features all the content from the original Stronghold Crusader but also adds new opponents to battle and the Warchest trail.

Stronghold Crusader Extreme is a brand new version of Stronghold Crusader featuring new extreme features, such as the outposts and super attacks, which can be used in multiplayer and skirmish modes.  There are also new opponents to battle and the Extreme trail.

Section 3 – Multiplayer Help

Stronghold Crusader/Stronghold Crusader Extreme Multiplayer

If your computer is connected to a router/firewall you will need to forward the following Microsoft DirectPlay Firewall Ports to your computer in order to play Stronghold Crusader or Stronghold Crusader Extreme multiplayer.
 

 Outbound Inbound
Initial TCP Connection 47624 47624
Subsequent TCP 2300-2400 2300-2400
Subsequent UDP 2300-2400 2300-2400


If you require help configuring your router please visit portforward.com which has an excellent range of guides for over 1000 routers.

http://www.portforward.com/english/routers/port_forwarding/routerindex.htm

Windows Firewall

1.    Open Control Panel > Windows Firewall
2.    Select the Exceptions tab
3.    If Stronghold Crusader and Stronghold Crusader Extreme are not in the list click Add Program.
4.    Select Stronghold Crusader and click OK
5.    Repeat the process for Stronghold Crusader Extreme
6.    Click OK to close the Window Firewall panel

Section 4 – Additional Features

Radar Map

While in-game pressing ALT-R changes the radar map to show the enemies all in one colour and the player and allies in another colour.

Extreme Powers Around Lord
 
Within Extreme Multiplayer Play Options, you can active ‘Extreme Powers Around Lord‘.  This option enables the player to spawn troops using the Tactical Aid Bar within the radius of the lord rather than just within the radius of your Keep.
